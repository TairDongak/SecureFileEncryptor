# SecureFileEncryptor

Приложение для защиты данных в папках путем рекурсивного шифрования (AES-256).

## Постановка задачи
Реализовать защиту данных пользовательских папок и файлов, находящихся в подпапках, путем шифрования. Реализация выполнена на C++ с использованием библиотеки Qt и паттерна проектирования Singleton.

## Инструментарий
* **Язык:** C++
* **Библиотеки:**
    * Qt Core (работа с файловой системой, QDirIterator).
    * tiny-AES-c (реализация алгоритма AES-256).
* **Среда разработки:** Visual Studio + Qt Visual Studio Tools.

## Описание паттерна Singleton
В проекте используется порождающий паттерн **Одиночка (Singleton)**.

**Назначение:**
Паттерн гарантирует, что у класса `CryptoManager` есть только один экземпляр, и предоставляет к нему глобальную точку доступа. Это необходимо для контроля над процессом шифрования, чтобы никакой другой код не мог подменить экземпляр класса.

**Реализация в проекте:**
Согласно рекомендациям, в классе `CryptoManager` выполнены следующие шаги:
1.  Добавлено приватное статическое поле (в реализации метода `Instance`) для хранения объекта.
2.  Объявлен статический метод `Instance()` для получения одиночки.
3.  Использована «ленивая инициализация» (объект создается при первом вызове).
4.  Конструктор класса сделан приватным, чтобы запретить создание экземпляров через `new`.

Также удалены конструктор копирования и оператор присваивания для предотвращения клонирования объекта.

## Сборка и запуск
1.  Клонируйте репозиторий.
2.  Откройте файл `SecureFileEncryptor.sln` в Visual Studio.
3.  Убедитесь, что установлено расширение Qt Visual Studio Tools.
4.  Соберите проект (Build -> Build Solution).
5.  Запустите приложение.

## Руководство пользователя
1.  При запуске введите полный путь к папке, которую нужно обработать.
2.  Введите пароль. Запомните его! Без него файлы восстановить невозможно.
3.  Выберите действие:
    * `1` — Зашифровать все файлы в папке и подпапках.
    * `2` — Расшифровать файлы.
